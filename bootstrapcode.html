19. Customization with Sass

Instructions:
  1. Initialize npm and install Bootstrap and Sass: 
    npm init -y
npm install bootstrap@5.3.3 sass
  2. Create scss/custom.scss with: 
    $primary: #ff5722;
$border-radius: 2rem;
@import "../node_modules/bootstrap/scss/bootstrap";
  3. Compile SCSS to CSS: 
    npx sass scss/custom.scss css/custom.css
  4. Link css/custom.css in your HTML instead of the CDN for custom Bootstrap styles.
